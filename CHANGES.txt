# platinfo change log

## platinfo 2.0.0 (not yet released)

- Change module name to "platinfo2" for sane co-existence with the "platinfo"
  (version 0.x) module.


## platinfo 0.15.1 (not yet released)

- Add `ia64_32` arch name for 32-bit builds on HP-UX/IA64 and `powerpc64` arch name for 64-bit builds on AIX/PowerPC.
- Issue 8: python3: TypeError: can't use a string pattern on a bytes-like object


## platinfo 0.15.0

- Python 3 support (thanks Sridhar Ratnakumar)

## platinfo 0.14.5 (20-Mar-2010)

- Add support for OpenBSD on x86_64 (thanks Jordan Earls).
